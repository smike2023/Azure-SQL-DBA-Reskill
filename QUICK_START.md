## Step 2: Set Up Your Lab (30-60 minutes)

Choose **one** setup option to start:

### Option A: Azure Free Tier (Easiest)
1. Create new email address for Azure
2. Sign up at portal.azure.com
3. Create free SQL Database
4. Test connection with Query Editor

### Option B: Local SQL Server (Most Control)
1. Download SQL Server 2022 Developer Edition
2. Install SQL Server and SSMS
3. Create test database
4. Practice basic T-SQL

### Option C: Docker Container (Modern)
1. Install Docker Desktop
2. Run SQL Server container
3. Connect with SSMS
4. Practice container operations
